📄 Import from CSV / Excel

Export from EliseAI or save your Excel sheet as CSV (File → Save As → CSV). New records are added, existing ones are matched by unit+date and updated — no duplicates.

📄 CSV Format Guide
Tours CSV columns (any order):
name, date, time, unit, type (sgt/guided), contact, notes
Moves CSV columns:
unit, resident, date, type (in/out), time, total, elevator (yes/no), notes
Maintenance CSV columns:
unit, issue, description, priority (high/medium/low), status (open/in-progress/resolved), reporter
Amenity Bookings CSV columns:
resident, unit, space, date, start, end, type (reservation/event), fee, notes
Tip: Column names are flexible — the importer matches common variations (e.g. "Prospect Name", "prospect_name", "Name" all map to name).
